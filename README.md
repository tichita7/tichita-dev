## Installation

Clone the repository:

```bash
git clone https://github.com/tichita7/tichita.dev.git
```

Move into project directory:

```bash
cd tichita.dev
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
